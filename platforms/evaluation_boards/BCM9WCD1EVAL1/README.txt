--------------------------------------------
BCM9WCD1EVAL1 - README
--------------------------------------------

The BCM9WCD1EVAL1 Evaluation Board is used to
evaluate 31-pin WICED modules. These modules
all have a 0.1" pin pitch. 

Modules supported by BCM9WCD1EVAL1 include:
  - BCM943362WCD2
  - BCM943362WCD4
  - BCM943362WCD6
  - BCM943362WCD8
  - BCM9WCDUSI09
