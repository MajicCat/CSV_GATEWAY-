Detailed Description

codec_interface_t: each supported codec must provide this interface.

Field Documentation

get_decoded_output_size

This returns the size of the array with decoded pcm samples from the decoder in bytes.

This is required when the size of the output is required and the decoder cannot just yet be started after configuring the codec.

wiced_codec_type_t type

Indicates the codec type.

Should be filled by the codec.
