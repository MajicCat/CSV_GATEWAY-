Detailed Description

Initialisation/Deinitialisation of WICED and device configuration functions.

Functions to resume WICED after the deep-sleep.

Platform is not necessary to support deep-sleep mode.

wiced_result_t wiced_deinit ( void  )

De-initialises the WICED system.

This function de-initialises the WICED system by :

  • bringing down all network interfaces
  • deleting all packet pools
  • tearing down the event thread
  • powering down the WLAN chip
Returns
wiced_result_t

wiced_result_t wiced_init ( void  )

Initialises the WICED system.

This function sets up the system by :

  • initialising the platform interface
  • initialising the RTOS & Network Stack
  • initialising the WLAN driver and chip
  • starting the event processing thread
