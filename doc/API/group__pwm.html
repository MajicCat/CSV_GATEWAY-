wiced_result_t wiced_pwm_init ( wiced_pwm_t  pwm,
uint32_t  frequency,
float  duty_cycle 
)

Initialises a PWM pin.

Prepares a Pulse-Width Modulation pin for use. Does not start the PWM output (use wiced_pwm_start).

Parameters
pwm: the PWM interface which should be initialised
frequency: Output signal frequency in Hertz
duty_cycle: Duty cycle of signal as a floating-point percentage (0.0 to 100.0)
Returns
WICED_SUCCESS : on success.
WICED_ERROR : if an error occurred with any step
