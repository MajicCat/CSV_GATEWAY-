Detailed Description

Functions to communicate with the system monitor.

Functions to initialise WICED in a more modular way.

wiced_init() = wiced_core_init() + wiced_wlan_connectivity_init() wiced_deinit() = wiced_core_deinit() + wiced_wlan_connectivity_deinit()
